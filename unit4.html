Lesson#6 C.F.U

What does it mean to put the most specific case first? Why is it important to put the most specific case first? What types of errors does it help avoid?

When creating an if-else-if statement, it is really important to make the most specific condition the first condition. When you run your code, the code is read from top to bottom. Therefore, putting the most specific case is important because it will be ran first. In addition, by putting the most specific case first, it will prevent errors such as the whole code not working or an error where the more general commands work and the specifics don't. All in all, putting the most specific case first means to put the code with the most specific and selective value. For example, if one statement has a value of ==15 and the other statement is <=100, the ==15 should be first since it is more specific.
